# Branch 기초 명령어

* 브랜치 생성

  ```bash
  $ git branch {브랜치이름}
  ```

* 브랜치 목록

  ```bash
  $ git branch
  * master
    test
  ```

* 브랜치 이동

  ```bash
  $ git checkout test
  Switched to brach 'test'
  (test) $
  ```

* 브랜치 생성 및 이동

  ```bash
  $ git checkout -b test2
  ```

* 브랜치 병합

  ```bash
  $ git merge test2
  ```

* 브랜치 삭제

  ```bash
  $ git branch -d test2
  Deleted branch test2 (was 3888136).
  ```
